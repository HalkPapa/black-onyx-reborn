## 🎯 技術仕様

### **開発環境**
- **エンジン**: Unity 2022.3.62f1 LTS
- **言語**: C# (.NET Standard 2.1)
- **プラットフォーム**: Windows, macOS, Linux
- **アーキテクチャ**: Manager Pattern + Singleton Pattern

### **プロジェクト構成**
```
Assets/Scripts/
├── Managers/              # 中央管理システム
│   ├── GameManager.cs     # ゲーム統合管理
│   ├── AudioManager.cs    # 音声システム (16種SE + フロア別BGM)
│   └── UIManager.cs       # UI統合管理
├── Dungeon/               # ダンジョンシステム
│   ├── BlackOnyxDungeonManager.cs    # メインダンジョン (8フロア)
│   ├── DungeonManagerBridge.cs       # 新旧システム統合
│   └── BlackOnyxFloor.cs             # フロア管理
├── Combat/                # 戦闘システム
│   └── CombatManager.cs   # 1984年準拠戦闘・成長システム
├── Items/                 # アイテムシステム
│   ├── ItemManager.cs     # オリジナル準拠アイテム管理
│   ├── SpecialItemManager.cs # 特殊アイテム・エンディング
│   └── FireCrystalSystem.cs  # ファイヤークリスタル連動
├── Integration/           # クロスゲーム統合
│   └── CrossGameIntegration.cs # 4ゲーム統合システム
├── Effects/               # エフェクトシステム
│   └── UIEffectManager.cs # 華やかな演出効果
├── Optimization/          # 最適化システム
│   └── PerformanceManager.cs # 次世代パフォーマンス管理
└── SaveSystem/            # セーブシステム
    └── BlackOnyxSaveManager.cs # 3層セーブシステム
```
